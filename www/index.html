d3 Disco - An example of interfacing shiny and d3.js

This page tries to give a minimal example of how to interface R and d3.js using the shiny package. It contains a custom output and a custom d3 based plot function. The shiny server instance is generating circles at random timepoints and with random attributes (x, y, radius, colour and exit function) and sends these to the client, where they are fed into the d3 function.

For more involved applications it would probably be better to define a local datamodel on the client, where data from the server is added. The datamodel would then trigger a change event which could be picked up by the d3 plotting function. This setup makes it easier for several unrelated d3 plots to depend on the same data and update automatically when new data arrives.
